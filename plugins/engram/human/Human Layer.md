# Human Layer — Personal Truth

This layer stores the **personal truth** about the human(s) driving the brain — *not* code, *not*
project facts. It's what makes every project "know you the same way." Supports **multiple personas**
(e.g. the same person as a business operator vs. on a personal project) and **multiple humans**.

> Adopted into each project by `brain-sync`. Contains nothing project-secret (Layer D).
>
> **In the public release this layer is a template.** The `profiles/` folder ships with an example
> (`primary.md`) full of placeholders — fill it in with your own truth locally. It lives on your machine
> and is never uploaded, synced, or sent anywhere by Engram.
